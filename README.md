# Supply Chain EDA Project

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a supply chain dataset to understand shipment patterns, cost distribution, and logistics performance.

## 🎯 Objective
- Analyze shipment patterns
- Understand cost factors
- Improve logistics efficiency

## 📊 Key Insights
- Freight cost increases with weight and quantity
- Air shipment is most frequently used
- Few vendors dominate shipments

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Visualizations
- Bar charts
- Boxplots
- Histograms
- Scatter plots
- Correlation heatmap

## 🚀 Conclusion
The analysis helps in optimizing shipment strategies, reducing costs, and improving supply chain efficiency.
